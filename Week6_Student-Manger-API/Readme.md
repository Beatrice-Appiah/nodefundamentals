1.  Introduction to restful API's
Restful API is a backend service that allows application communicates with each other over the internet using HTTP protocols. It is based on representational state transfer (REST) architecture, which uses standard HTTP methods such as GET, POST, PUT, DELETE to perform operations on resources.
Returns data usually in JSON

- Why Rest it is important
Because it's Simple, Fast and widely used.

- Basic REST Methods
GET: Retrieve data from server.
POST: Send data to server to create a new resource.
PUT: Update existing resource on server.
DELETE: Remove resource from server.
PATCH: Partially update existing resource on server.
